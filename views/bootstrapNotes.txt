NAVBARS
1).html tag "nav" can be used to create a navbar, which will contain the nested menu items
	-"navbar" class applies default styling
2).html tag "ul" can be nested in the "nav" tag to contain menu items
	-"nav" class applies highlighting and stacks the nested items, with a styling
	-"navbar-nav" class applies some styling, resizability and stacks horizontally
3).html tag "div" can be nested within the "nav" tag to hold a menu header
	-"navbar-header" class applies text styleing and highlighting
4).html tag "a" can be nested in the "div" tag to become a menu title
	-"navbar-brand" class applies defualt styling.
